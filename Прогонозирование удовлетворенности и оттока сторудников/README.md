# HR-аналитика: Прогнозирование удовлетворённости и оттока сотрудников  
## Компания «Работа с заботой»

**Статус:** завершён  

---

## Кратко о проекте
HR-аналитики компании «Работа с заботой» разрабатывают модели для решения двух ключевых задач:
1. Прогнозирование уровня удовлетворённости сотрудников (регрессионная задача)
2. Прогнозирование вероятности увольнения сотрудника (классификационная задача)

**Почему это важно для бизнеса:**
- Удовлетворённость работой напрямую влияет на отток сотрудников.
- Внезапные увольнения, особенно ключевых специалистов, несут риски для компании.
- Прогнозирование позволяет принимать превентивные меры для удержания персонала.

---

## Цель
- Прогнозировать уровень удовлетворённости сотрудников (задача регрессии)  
- Прогнозировать вероятность увольнения сотрудника (задача классификации)  
- Повысить удержание сотрудников через своевременные действия  

---

## Данные
- **Признаки:**
  - `id` — уникальный идентификатор сотрудника
  - `dept` — отдел (sales, hr, technology, purchasing, marketing)
  - `level` — уровень должности (junior, middle, senior)
  - `workload` — загруженность (low, medium, high)
  - `employment_years` — стаж работы (в годах)
  - `last_year_promo` — повышение за последний год (yes/no)
  - `last_year_violations` — нарушения трудового договора (yes/no)
  - `supervisor_evaluation` — оценка руководителя (1-5)
  - `salary` — зарплата
  - `job_satisfaction_rate` — удовлетворённость (0-1) — целевой признак для задачи 1
  - `quit` — уволился ли сотрудник (yes/no) — целевой признак для задачи 2

- **Объем данных:**
  - Обучающая выборка: 4000 сотрудников
  - Тестовая выборка: 2000 сотрудников

---

## Подход
- Загрузка и предобработка данных  
- Анализ и исследование данных (EDA)  
- Создание и настройка моделей для предсказания удовлетворённости и увольнений  
- Обучение моделей с настройкой гиперпараметров  
- Оценка качества моделей с использованием метрик SMAPE и ROC-AUC  

---

## Модели
### Задача 1: Предсказание удовлетворённости
- GradientBoostingRegressor (лучшая модель)
- LinearRegression
- Lasso
- DecisionTreeRegressor

### Задача 2: Предсказание увольнения
- GradientBoostingClassifier (лучшая модель)
- RandomForestClassifier
- LogisticRegression
- SVC

**Метрики:**
- SMAPE (для регрессии)
- ROC-AUC, F1-score, accuracy, precision, recall (для классификации)

---

## Результаты
### Задача 1: Прогнозирование удовлетворённости
- **Лучшая модель:** GradientBoostingRegressor  
- **Гиперпараметры:**  
  - `max_depth`: 6  
  - `n_estimators`: 100  
  - `learning_rate`: 0.1  
- **SMAPE (Test):** 11.99%  
- **SMAPE (DummyRegressor):** 38.26%

### Задача 2: Прогнозирование увольнения
- **Лучшая модель:** GradientBoostingClassifier  
- **Гиперпараметры:**  
  - `learning_rate`: 0.05  
  - `max_depth`: 4  
  - `min_samples_leaf`: 8  
  - `n_estimators`: 300  
- **ROC-AUC (CV):** 0.9224  
- **ROC-AUC (Test):** 0.91

---

## Бизнес-выводы
### Задача 1: Удовлетворённость
- **Факторы удовлетворённости:**
  - Высокая оценка руководителя сильно коррелирует с удовлетворённостью (0.76)
  - Зарплата и уровень должности влияют на удовлетворённость
  - Неожиданно: сотрудники с нарушениями могут быть более удовлетворены

### Задача 2: Увольнение
- **Факторы увольнения:**
  - Junior-сотрудники увольняются чаще всего
  - Низкая загруженность увеличивает риск увольнения на 100% по сравнению с высокой
  - Отсутствие повышения: только 0.9% уволившихся получили повышение
  - Нарушения дисциплины: 37.8% уволившихся имели нарушения
  - Короткий стаж: наибольший риск в первые 1-2 года работы
  - Низкие оценки руководителя увеличивают риск увольнения

## Рекомендации для компании «Работа с заботой»
1. **Для снижения оттока junior-сотрудников:**
   - Внедрить программу наставничества
   - Улучшить процесс адаптации новых сотрудников
   - Обеспечить адекватную загруженность с первых месяцев работы

2. **Для повышения удовлетворённости:**
   - Обучить руководителей навыкам обратной связи
   - Внедрить систему регулярных performance review
   - Создать прозрачную систему карьерного роста

3. **Для превентивного удержания:**
   - Внедрить систему алертов по сотрудникам с высоким риском увольнения
   - Разработать индивидуальные планы развития для ключевых специалистов
   - Создать программу retention для high-performers

4. **Организационные изменения:**
   - Пересмотреть распределение нагрузки между сотрудниками
   - Усилить работу с дисциплинарными нарушениями
   - Создать систему раннего выявления неудовлетворённых сотрудников
